Overview

 ImageShow is a modern GPU-accelerated image and slideshow viewer app. Slideshow of images is generated at run-time with effects, transitions and background music as configured.

Features

Image Explorer

ImageShow has a built-in image explorer for quickly browsing images that are shown as thumbnails. The following image formats are currently supported
  • Windows Bitmap Format (.bmp, .dib)
  • DirectDraw Surface (.dds)
  • Digital Negative (.dng)
  • Graphics Interchange Format (.gif)
  • High Efficiency Image File (.heic, .heif)
  • Icon File (.ico)
  • Joint Photographic Experts Group (.jpe, .jpeg, .jpg)
  • JPEG Extended Range format (.jxr)
  • JPEG File Interchange Format (.jfi, .jfif, .jif)
  • Portable Network Graphics (.png)
  • Tagged Image File Format (.tif, .tiff)
  • Windows Media Photo (.wdp)
Images can also be flipped and viewed one at a time in the app's window size. The image is automatically scaled-up or scaled-down to fit the window size while maintaining the aspect ratio. In flip view mode, useful information about the image can be seen easily.

Frequently Asked Questions (FAQ)
  • Why can't I view High Efficiency Image files (.heic, .heif)?

    • ImageShow relies on Windows to support decoding of these formats. Please install the "HEIF Image Extensions" app published by Microsoft from the Store.
