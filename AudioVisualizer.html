Overview

 AudioVisualizer is a real-time audio analyzer and visualizer app with a built-in player. It can analyze and visualize audio from its own player, any audio playback device (loopback mode) and any audio capture device such as a microphone.

Features

  • Player

    AudioVisualizer has an built-in audio player that currently supports the following formats

    Uncompressed

    • Linear Pulse-Code Modulation (.wav)

    Lossless compression

    • Apple Lossless Audio Codec (.m4a)
    • Free Lossless Audio Codec (.flac)
    • Windows Media Audio Lossless (.wma)

    Lossy compression

    • Adaptive Multi-Rate Audio Codec (.amr)
    • Advanced Audio Coding (.m4a, .m4r, .aac)
    • Audio Data Transport Stream (.adt, .adts)
    • Audio Codec 3 (.ac3)
    • Enhanced Audio Codec (.ec3)
    • Matroska Audio (.mka)
    • MPEG-1 Audio Layer III or MPEG-2 Audio Layer III (.mp3)
    • MPEG-2 Audio (.mpa)
    • MPEG-4 Audio (.m4a, .m4r)
    • Ogg Vorbis (.oga, .ogg)
    • Opus (.opus)
    • Windows Media Audio Lossy (.wma)

    Multimedia Containers

    • MPEG-4 Part 14 (.mp4)

  • Audio Explorer

    Audio explorer allows quick browsing of audio files presented in a folders and files view. Files are shown with filename, album and artist info. Audio files can be selected and added to the default playlist.

  • Playlist Manager

    AudioVisualizer currently has a simple playlist manager that helps to re-order or delete items from the default playlist.
